1. Output & Variabel

Python menggunakan print() untuk menampilkan teks ke layar.

print("Halo, dunia!")
nama = "Marvel"
umur = 20
print("Nama:", nama)
print("Umur:", umur)

Nama: Marvel
Umur: 20

2. Operasi Aritmetika

Python mendukung operasi seperti +, -, *, /, %, dan ** (pangkat).

a = 10
b = 3
print(a + b)
print(a ** b)
13
1000

3. Konversi & Manipulasi Data

Gunakan fungsi bawaan seperti int(), str(), list(), dan set() untuk konversi tipe data.

angka_str = "42"
angka_int = int(angka_str)
print(angka_int + 8)

data_list = [1, 2, 3, 3, 2]
print(set(data_list))    # hapus duplikat
print(tuple(data_list))  # ubah ke tuple
50
{1, 2, 3}
(1, 2, 3, 3, 2)

4. Manipulasi Input Pengguna

Input bawaan Python selalu berupa string. Kita dapat ubah ke huruf kecil, besar, atau format lain.

teks = input("Masukkan kalimat: ")

print(teks.lower())  # jadi huruf kecil semua
print(teks.upper())  # jadi huruf besar semua
print(teks.title())  # kapital tiap kata

5. Manipulasi List

List di Python sangat fleksibel ‚Äî bisa ditambah, dihapus, diurutkan, atau diakses per elemen.

buah = ["apel", "pisang", "mangga"]

buah.append("jeruk")    # tambah item
buah.remove("pisang")   # hapus item
buah.sort()             # urutkan
print(buah)
print(buah[0])          # akses item pertama
['apel', 'jeruk', 'mangga']
apel

6. Dictionary (Kamus Data)

Dictionary menyimpan data dalam format pasangan key: value.

orang = {
    "nama": "Marvel",
    "umur": 20,
    "kota": "Jakarta"
}

print(orang["nama"])
orang["umur"] = 21
orang["hobi"] = "coding"
print(orang)
Marvel
{'nama': 'Marvel', 'umur': 21, 'kota': 'Jakarta', 'hobi': 'coding'}

7. Kombinasi Input, List, dan Dictionary

Contoh mengolah input pengguna menjadi list, lalu membentuk dictionary sederhana.

buah_input = input("Masukkan nama buah (pisahkan spasi): ").split()
harga_input = list(map(int, input("Masukkan harga tiap buah: ").split()))

data_buah = dict(zip(buah_input, harga_input))
print(data_buah)
Masukkan nama buah: apel jeruk mangga
Masukkan harga tiap buah: 10000 8000 12000

{'apel': 10000, 'jeruk': 8000, 'mangga': 12000}
